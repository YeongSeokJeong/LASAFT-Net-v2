Demonstration: V2_large¶

Comparison with LaSAFT-Net-v1 (a.k.a. LaSAFT+GPoCM)

In this section, we provide audio samples extracted by using LASAFT-Net-v2. We use the same tracks used in the demonstartion of LaSAFT-Net-v1. LaSAFT-Net-v1 is our previous version, proposed in the paper “Latent Source Attentive Frequency Transformation for Conditioned Source Separation”.

Official Github Repo : ws-choi/Conditioned-Source-Separation-LaSAFT
